# EMOTION_MAP_SPEC.md

## Objectif
Afficher une cartographie émotionnelle avancée V1 basée sur les scans réels du chien.

## Éléments à afficher
- Top états probables / validés
- Heatmap horaire
- Top déclencheurs
- Top contextes
- Évolution dans le temps
- Nombre de scans validés
- Nombre de corrections
- Tendances récurrentes
- Conseils personnalisés si assez de données

## Sources
- scans
- scan_state_scores
- recurring_patterns
- user_feedback_events

## Règles
- Affichage utile et compréhensible
- Pas de prétention clinique
- Mise en avant des récurrences et patterns
