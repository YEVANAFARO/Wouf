# REFERRAL_AND_MONETIZATION_SPEC.md

## Auth / inscription
Champs minimum :
- email
- mot de passe
- code parrain optionnel
- téléphone optionnel
- confirmation d'âge ou mois + année
- consentement CGU / confidentialité / audio

## Parrainage
### Champs
- referral_code
- referred_by
- referral_count
- founder_status
- beta_priority_score

### Paliers initiaux
- 1 filleul : badge fondateur
- 3 filleuls : priorité bêta
- 5 filleuls : avantage premium en attente
- 10 filleuls : VIP fondateur

## Plans
### Free
- 3 scans / jour
- 1 chien
- 7 jours d'historique
- analyse rapide

### Plus fondateur
- scans illimités
- empreinte sonore en apprentissage
- historique étendu
- accès anticipé à certaines fonctions

### Pro fondateur
- cartographie avancée
- support prioritaire
- futures fonctions premium
- multi-chiens si prêt, sinon mention "bientôt"
