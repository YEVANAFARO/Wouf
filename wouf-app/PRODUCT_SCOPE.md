# PRODUCT_SCOPE.md

## Ce qu'on implémente maintenant
- Compte utilisateur simple
- Parrainage dès l'inscription
- Statut fondateur
- Profil chien
- Enregistrement audio
- Détection bark / non-bark
- Features audio
- Moteur d'interprétation probabiliste
- Questions contexte + langage corporel
- Validation / correction
- Historique
- Empreinte sonore avancée V1
- Cartographie émotionnelle avancée V1
- Paywall Free / Plus / Pro
- Shop teaser "À venir"
- Bêta fermée rapide

## Ce qu'on ne fait pas maintenant
- Traduction exacte du chien
- Modèle de recherche profond entraîné sur un énorme dataset
- Vraie boutique e-commerce
- Loterie réelle
- Intégration partenaires réelle
- Gamification lourde
- Sortie App Store publique complète cette semaine
- Multi-chiens avancé si cela ralentit le MVP

## Positionnement retenu
WOUF est un interprète comportemental canin assisté par IA qui s'améliore pour chaque chien.
